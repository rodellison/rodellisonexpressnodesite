MusicManParmFixer MusicManParmFixer is a simple express/bootstrap app to allow quick updates of Artists or Venue values into an AWS DynamoDB table. The values stored by this app are leveraged by the MusicMan Alexa Skill. Effectively, the app provides a quick and dirty means to provide some 'corrected' values so that if someone uses the Alexa Skill Music Man and says.. where is Rianna playing, the skill can do a quick lookup with the value provided by the customer, and do a swap if a corrected value is available. e.g it can quickly swap the bad value of Rianna out, and instead use the correct value "Rihanna". Lastly, for values successfully updated into the AWS DynamoDB, a seperate AWS SNS topic is provided a message, which is then sent via email to a subscriber.
